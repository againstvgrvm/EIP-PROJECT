# BuildConnect Benin

BuildConnect Benin est une plateforme digitale haut de gamme dédiée au secteur du BTP au Bénin. Elle connecte les clients, les prestataires et les fournisseurs de matériaux dans un écosystème axé sur l'excellence et la transparence.

## Fonctionnalités principales

- **Flux d'actualités dynamique** : Visualisation des projets en cours et des offres de matériaux (style Instagram/Premium).
- **Inscription segmentée intelligente** : Parcours utilisateur avec champs dynamiques adaptés selon le profil choisi (Client, Prestataire ou Fournisseur).
- **Interface Apple-Style** : Design minimaliste, typographies fortes, animations fluides (Framer Motion) et charte graphique luxueuse (Marron, Rouge, Or).
- **Recherche et Filtrage** : Localisation des experts et des ressources par zone géographique et spécialité.
- **Zéro Backend Requis** : Utilisation du stockage local et architecture moderne sans compilation (No-Build).

## Guide de Déploiement

Le projet est prêt pour une exportation immédiate.

### Option 1 : Vercel (Recommandé)
1. Poussez votre dossier sur un dépôt GitHub.
2. Connectez votre compte à [Vercel](https://vercel.com).
3. Importez le projet. Le déploiement est automatique.

### Option 2 : Netlify
1. Glissez simplement le dossier racine du projet sur [Netlify Drop](https://app.netlify.com/drop).

### Option 3 : GitHub Pages
1. Poussez le code sur GitHub.
2. Dans `Settings > Pages`, activez le déploiement sur la branche `main`.

## Développement Local

```bash
# Avec Node.js (facultatif mais recommandé pour le serveur local)
npm start
```

## Technologies
- **React 19** : Interface utilisateur réactive.
- **Tailwind CSS** : Design système flexible.
- **Framer Motion** : Animations de micro-interactions.
- **Lucide Icons** : Iconographie moderne.
- **TypeScript** : Code robuste et maintenable.
